# Predicting-Cognitive-Performance-from-Sleep-Metrics
An end-to-end data pipeline that collects, stores, and analyzes sleep quality data to predict cognitive performance scores. Rather than relying on simple sleep duration measures, we examine multiple sleep dimensions: REM sleep, deep sleep, sleep latency, wake episodes - alongside lifestyle covariates to build interpretable regression models.
